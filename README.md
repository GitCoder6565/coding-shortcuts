# 🚀 Ultimate Developer Shortcuts Guide
**Organized from Most to Least Useful for Daily Coding**
---
## 🎯 **TIER 1: Essential Daily Shortcuts** (Use these 100+ times/day)
### **VS Code - Core Navigation & Editing**
| Shortcut | Action | Why It's Critical |
|----------|--------|-------------------|
| `Ctrl + P` | Quick file open | Fastest way to navigate projects |
| `Ctrl + Shift + P` | Command palette | Access any VS Code command |
| `Ctrl + D` | Select next occurrence | Multi-cursor editing magic |
| `Ctrl + /` | Toggle line comment | Comment code instantly |
| `Ctrl + Shift + K` | Delete line | Remove lines without selecting |
| `Alt + ↑/↓` | Move line up/down | Reorganize code quickly |
| `Ctrl + Space` | Trigger IntelliSense | Auto-complete suggestions |
| `Ctrl + .` | Quick fix | Fix errors/refactor instantly |
| `F2` | Rename symbol | Refactor variable/function names |
| `Ctrl + Click` | Go to definition | Navigate to source code |
| `Alt + ← / →` | Navigate back/forward | Move through edit history |
| `Ctrl + B` | Toggle sidebar | Maximize screen space |
### **Windows OS - Window Management**
| Shortcut | Action | Why It's Critical |
|----------|--------|-------------------|
| `Win + D` | Show/hide desktop | Clear workspace instantly |
| `Win + Tab` | Task view | See all open windows |
| `Alt + Tab` | Switch windows | Fast app switching |
| `Win + ← / →` | Snap window left/right | Split screen productivity |
| `Win + Shift + S` | Screenshot tool | Capture anything quickly |
| `Win + V` | Clipboard history | Access previous copies |
| `Win + L` | Lock computer | Security when stepping away |
| `Ctrl + Shift + Esc` | Task Manager | Kill frozen apps |
### **Git - Essential Commands**
| Command/Shortcut | Action | Why It's Critical |
|------------------|--------|-------------------|
| `git status` | Check repo state | Know what's changed |
| `git add .` | Stage all changes | Prepare for commit |
| `git commit -m "msg"` | Commit with message | Save your work |
| `git push` | Push to remote | Share your code |
| `git pull` | Pull latest changes | Stay synchronized |
| `git log --oneline` | View commit history | See project timeline |
| `git diff` | See unstaged changes | Review before committing |

## 🧠 **Pro Tips for Mastering Shortcuts**
### **Learning Strategy**
1. **Week 1**: Master TIER 1 (10 shortcuts)
2. **Week 2**: Add TIER 2 (15 shortcuts)
3. **Week 3**: Incorporate TIER 3 (10 shortcuts)
4. **Week 4**: Experiment with TIER 4
### **Practice Techniques**
- **Print this guide** and keep it visible
- **Force yourself** to use shortcuts instead of mouse
- **Track usage** - note which ones you use most
- **Customize** - create your own shortcuts in VS Code
### **VS Code Customization**
- Open keyboard shortcuts: `Ctrl + K, Ctrl + S`
- Search for any command and assign custom shortcuts
- Export/import shortcuts across machines
### **Productivity Multipliers**
- Combine shortcuts: `Ctrl + D` → `Ctrl + D` → `Ctrl + D` = select 3 occurrences
- Use multi-cursor + shortcuts for bulk edits
- Chain Git commands: `git add . && git commit -m "msg" && git push`
